# StarUML Design Patterns

In this project we present the well-known OOP design patterns from the [GoF Book]( https://en.wikipedia.org/wiki/Design_Patterns) in [StarUML](http://staruml.io)

EcmaScript 6 implementation: [es6-design-patterns](https://github.com/loredanacirstea/es6-design-patterns)

For the Javascript implementation see: [js-design-patterns](http://loredanacirstea.github.io/js-design-patterns)

Images were generated by [StarUML](http://staruml.io/) from the .mdj file.

Everyone is welcome to improve on this and provide implementation in various programming languages.
